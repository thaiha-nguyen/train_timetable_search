## How to run the code
### Prepare
* Installation
```shell
# selenium
pip pip install selenium

# beautifulsoup 
pip install beautifulsoup4

# chrome driver
brew install --cask chromedriver
```

### Run code
```shell
python main.py --start <出発駅> --end <到着駅>

```
* Example: `python main.py --start 東京 --end 盛岡`
